*---------------git--------------* *-----------------configuring git---------------------* git config --global user.name "My Name" git config -global user.email "user@email234.com" git config --list *-------------------Clone & status----------------------------* Clone -> cloning a repositry on your local machine git clone <-some link-> Status -> display the status of the code git status We have four types of status 1. untracked -> new file that git doesn't track yet 2. modified -> changed 3. staged -> file is ready to be committed 4. unmodified -> unchanged *-----------------Add & Commit-------------------------* 1. add adds new or changed file in your working directory to the git staging area git add <-file name-> git add . //This command adds all files and directories in the current working directory and all its subdirectories to the staging area. It essentially adds everything recursively. 2.Commit it is the record of change git commit -m "some messeges" *-------------------push command -------------------------*
